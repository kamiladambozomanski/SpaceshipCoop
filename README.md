# Spaceship game

Made as a short prototype for 2 potential ideas, this idea wasn't used.

Focuses on asymmetrical controls and roles in a space enviorment including enemies

> This is a prototype game, the gameplay is to show an idea

## PC Player
- Top down controller
- 2D view of the ship
- Use terminals to access different turrets/utilities
- Defend ship directly

## Controller player
- The pilot flies through space
- Positions ships through dangers such as
- - Asteroid fields
  - Planet Gravity
